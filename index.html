Webpage Snapshot Bookmarklet

This is a simple bookmarklet that captures any webpage and save as png image using HTML5 technologies.

Under the hood it uses Drag and Drop to Bookmark bar to capture the webpage to canvas, then we convert the content inside this canvas element to image file.
